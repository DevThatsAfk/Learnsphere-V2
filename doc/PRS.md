# LearnSphere — PRS (Final)

## Vision
Help serious students build daily study consistency and understand effort vs outcomes — without pressure, rankings, or opaque AI.

## Primary User (v1)
- College student (18–24)
- Mobile-first, laptop secondary
- Wants structure, clarity, and feedback

## Core Non-Negotiable Daily Action
👉 Start and complete **at least one focused study session per day**

Everything else supports this.

## v1 Feature Set (Locked)
- Subjects
- Tasks + checklists
- Study sessions (active time only)
- Consistency calendar
- Neglect detection
- Exams & marks entry
- Lightweight review & flashcards

## Explicit Non-Goals
- Predictions
- Leaderboards
- Gamification
- Surveillance (keystrokes, screen tracking)

## Success Metrics
- ≥3 active days/week
- Reduced neglected subjects
- Clear weakest subject visibility

END.
