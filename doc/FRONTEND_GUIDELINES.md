# Frontend Rules — Non-Negotiable

## Rendering Rules
- Backend is the only source of truth
- No derived analytics on frontend
- No mock data in final build

## Every Screen MUST Have
- Loading state
- Empty state
- Error fallback

## UX Tone
- Calm
- Informational
- No red warnings
- No judgement language

## State Management
- AuthContext
- API Client abstraction
- Typed responses only
