# User Journey — LearnSphere

## 1. Onboarding & Setup
- **Goal:** Get ready to study.
- **Actions:**
  - User logs in (Auth).
  - User adds their current **Subjects** (e.g., Computer Science, Mathematics).
  - User adds **Exams** and their dates to the calendar.

## 2. Daily Planning
- **Goal:** Define what to work on today.
- **Actions:**
  - User creates **Tasks** for a specific subject.
  - User breaks tasks down into a **Checklist**.

## 3. Deep Work (The "Core Action")
- **Goal:** Focused study session.
- **Actions:**
  - User selects a subject and topic.
  - User starts a **Study Session** timer.
  - User completes the session and enters "active minutes".
  - User provides a quick **Reflection** and tags.

## 4. Feedback Loop
- **Goal:** Understand progress.
- **Actions:**
  - User checks the **Dashboard** for consistency (Active Days).
  - User views **Neglect Signals** to see which subject needs attention.
  - After an exam, user enters **Marks**.

## 5. Revision
- **Goal:** Long-term retention.
- **Actions:**
  - User enters the **Review Queue**.
  - User practices with **Flashcards** generated for their study topics.
  - User rates recall strength (Weak/Moderate/Strong).
